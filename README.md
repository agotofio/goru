# Art-Pub Gorynych — сайт (статичний)

## Як запустити локально
Відкрийте `index.html` у браузері.

## Як опублікувати на GitHub Pages
1. Завантажте **весь вміст** цього архіву в корінь репозиторію.
2. Settings → Pages → Branch: `main`, Folder: `/ (root)`.
3. Зачекайте 1–2 хв — зʼявиться посилання.

## Де правити меню
`data/menu.json` — всі позиції, ціни, ваги.

## Фото
- `images/menu/` — ваші скріни меню (оригінали)
- `images/gallery/` — фото для галереї (зараз плейсхолдери)
